Chapitre 5 : Équations différentielles

1. Définition

Une équation différentielle est une équation dans laquelle une fonction inconnue apparaît avec ses dérivées.

On s'intéresse ici aux équations du type :

\[ y' = a y \quad \text{où } a \in \mathbb{R} \]

2. Méthode de résolution

La solution générale de cette équation est :

\[ y(x) = C e^{a x} \quad \text{où } C \in \mathbb{R} \]

C’est une famille de fonctions exponentielles.

Exemple :

Résolvons \( y' = 2y \)

La solution générale est \( y(x) = C e^{2x} \)

Si on impose \( y(0) = 3 \), alors :

\[ 3 = C e^{0} \Rightarrow C = 3 \Rightarrow y(x) = 3e^{2x} \]

Exercices interactifs

Question 1 :
Résous l’équation \( y' = -5y \).

Correction :
La solution générale est \( y(x) = C e^{-5x} \), où \( C \in \mathbb{R} \)
Question 2 :
Trouve la solution de \( y' = 3y \) avec \( y(0) = 2 \).

Correction :
Solution générale : \( y(x) = C e^{3x} \)
\( y(0) = 2 \Rightarrow C = 2 \)
Donc \( y(x) = 2 e^{3x} \)
Question 3 :
Soit \( y(x) = 4e^{-2x} \). Vérifie qu'elle est solution de \( y' = -2y \).

Correction :
\( y'(x) = -8e^{-2x} \), et \( -2y(x) = -8e^{-2x} \)
Donc \( y' = -2y \), elle est bien solution.
Question 4 (quiz) :
Quelle est la solution générale de l’équation \( y' = ay \) ?



Réponse correcte : B)
La solution générale est \( y(x) = C e^{a x} \), avec \( C \in \mathbb{R} \)
