Chapitre 8 : Les équations différentielles – Introduction, résolution et applications

Description :

Dans ce chapitre, nous abordons les **équations différentielles**, qui sont utilisées pour modéliser une grande variété de phénomènes naturels, comme le mouvement des corps, la propagation des ondes, ou la croissance de populations. Nous commencerons par une introduction aux types d'équations différentielles, puis explorerons les méthodes de résolution et leurs applications pratiques dans des contextes réels.

1. Qu'est-ce qu'une équation différentielle ?

Une **équation différentielle** est une équation qui relie une fonction inconnue \( y(x) \) et ses dérivées. Elle exprime une relation entre les variations d'une fonction et sa propre dérivée.

Exemples d’équations différentielles :

  • Équation différentielle du premier ordre :
    \( \frac{dy}{dx} = 2x \)
  • Équation différentielle du second ordre :
    \( \frac{d^2y}{dx^2} + y = 0 \)

2. Types d'équations différentielles

  • Équations différentielles ordinaires (EDO) : Une équation qui dépend d'une seule variable indépendante (par exemple, \( x \)).
  • Équations différentielles partielles (EDP) : Une équation qui dépend de plusieurs variables indépendantes (par exemple, \( x \) et \( t \)).

3. Résolution des équations différentielles du premier ordre

Une équation différentielle du premier ordre prend la forme générale : \[ \frac{dy}{dx} = f(x, y) \] Pour résoudre cette équation, on peut utiliser la méthode de séparation des variables si l'équation est séparable, ou encore des méthodes d'intégration directe.

Exemple :

Résolvons l'équation différentielle \( \frac{dy}{dx} = x^2 \). La solution est : \[ y = \int x^2 \, dx = \frac{x^3}{3} + C \] Où \( C \) est une constante d'intégration.

4. Résolution des équations différentielles du second ordre

Les équations différentielles du second ordre prennent la forme : \[ \frac{d^2y}{dx^2} + p(x) \frac{dy}{dx} + q(x) y = 0 \] Ces équations peuvent être résolues par des méthodes variées comme la méthode des coefficients indéterminés ou la méthode de variation des paramètres.

Exemple :

Résolvons l’équation \( \frac{d^2y}{dx^2} + y = 0 \), qui est une équation différentielle homogène du second ordre : \[ \text{La solution générale est } y(x) = A \cos(x) + B \sin(x) \] Où \( A \) et \( B \) sont des constantes déterminées par les conditions initiales.

5. Applications des équations différentielles

Les équations différentielles sont utilisées pour modéliser une grande variété de phénomènes dans les sciences et l’ingénierie. Voici quelques exemples :

  • Le mouvement des ressorts : L’équation \( m \frac{d^2y}{dt^2} + k y = 0 \) modélise le mouvement d'un ressort (loi de Hooke).
  • La croissance des populations : L'équation \( \frac{dP}{dt} = rP(1 - \frac{P}{K}) \) modélise la croissance d'une population (modèle logistique).
  • Les circuits électriques : Les équations de Kirchoff pour les circuits RL, RC et RLC sont des exemples d’équations différentielles appliquées aux systèmes électriques.

Exercices interactifs

Question 1 :
Résolvez l’équation \( \frac{dy}{dx} = 3x^2 \).

Correction :
La solution est \( y(x) = x^3 + C \), où \( C \) est une constante d'intégration.
Question 2 :
Résolvez l’équation \( \frac{d^2y}{dx^2} + 4y = 0 \).

Correction :
La solution générale est \( y(x) = A \cos(2x) + B \sin(2x) \), où \( A \) et \( B \) sont des constantes.
Question 3 :
Résolvez l’équation \( \frac{dy}{dx} = y \).

Correction :
La solution est \( y(x) = Ce^x \), où \( C \) est une constante.
Question 4 (quiz) :
Quelle est la solution générale de l’équation \( \frac{d^2y}{dx^2} + 9y = 0 \) ?



Correction :
La bonne réponse est : A) \( y(x) = A \cos(3x) + B \sin(3x) \).
