Chapitre 1 : Analyse

1.1 – Notion de fonction

Une fonction associe à chaque valeur de x une seule valeur f(x). Ex : f(x) = x² + 2.

1.2 – Représentation graphique

On trace les points (x, f(x)). Une parabole correspond à x², une droite à ax + b, etc.

1.3 – Variations

  • Croissante si f(x₁) < f(x₂) quand x₁ < x₂.
  • Décroissante si f(x₁) > f(x₂).

1.4 – Limites et continuité

On écrit : limx→a f(x) = L.

1.5 – Dérivation

  • (xⁿ)' = n·xⁿ⁻¹
  • (u+v)' = u' + v'
  • (uv)' = u'v + uv'
  • (u/v)' = (u'v − uv') / v²

1.6 – Tangente

y = f′(x₀)(x − x₀) + f(x₀) est l’équation de la tangente au point x₀.

Exercices corrigés

Exercice 1 : f(x) = 3x² − 2x + 1
→ f′(x) = 6x − 2 ; f(1) = 2 ; f′(1) = 4
Tangente : y = 4(x − 1) + 2 = 4x − 2

Exercice 2 : f(x) = −x² + 4x
→ f′(x) = −2x + 4 → max en x=2 → croissante [0,2], décroissante [2,∞[

Exercice 3 : f(x) = 1/x
→ décroissante, asymptotes en x=0 et y=0

Exercice 4 : f(x) = (x² − 1)/(x − 1) = x + 1 si x ≠ 1
→ lim(x→1) = 2

Exercice 5 : f(x) = (x² + 1)/x
→ f′(x) = (x²−1)/x²

Chapitre 2 : Algèbre

2.1 – Identités remarquables

  • (a + b)² = a² + 2ab + b²
  • (a − b)² = a² − 2ab + b²
  • (a + b)(a − b) = a² − b²

2.2 – Équations du second degré

ax² + bx + c = 0 → Δ = b² − 4ac → racines : x = (−b ± √Δ)/2a

2.3 – Systèmes

On utilise substitution, combinaison ou égalisation.

2.4 – Inéquations

Utilisation du signe du produit ou quotient, tableaux de signes.

Exercices corrigés

Exercice 1 : (2x−3)²
= 4x² − 12x + 9

Exercice 2 : x² − 5x + 6 = 0
Δ = 25 − 24 = 1 → x = (5±1)/2 = 2 ou 3

Exercice 3 :
x + y = 5
2x − y = 4
→ y = 5 − x ; 2x − (5−x) = 4 → x = 3, y = 2

Exercice 4 : x² − 4 > 0
→ (x − 2)(x + 2) > 0
→ x ∈ (−∞, −2) ∪ (2, ∞)

Chapitre 4 : Probabilités

4.1 – Événements

Univers, événements élémentaires, événement certain, impossible.

4.2 – Probabilité

P(A) = nombre d’issues favorables / total

4.3 – Union / intersection

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

4.4 – Arbres de probabilité

Représentation graphique des cas possibles.

4.5 – Probabilité conditionnelle

P(B|A) = P(A ∩ B) / P(A)

Exercices corrigés

Exercice 1 : 3 boules rouges sur 5
→ P(rouge) = 3/5

Exercice 2 : A={2,4,6}, B={5,6}
→ A ∪ B = {2,4,5,6} → P = 4/6 = 2/3

Exercice 3 :
P(garçon) = 0.6 ; P(lunettes|garçon) = 0.3
→ P(garçon ∩ lunettes) = 0.6 × 0.3 = 0.18

Chapitre 5 : Algorithmique

5.1 – Notions de base

Un algorithme est une suite d’instructions (variables, tests, boucles).

5.2 – Variables et affectation

a ← 5
b ← a + 3 → b = 8

5.3 – Instructions conditionnelles

SI a > b ALORS
   AFFICHER a
SINON
   AFFICHER b
FINSI

5.4 – Boucles

POUR i DE 1 À 10 FAIRE
   AFFICHER i
FIN POUR

TANT QUE x > 0 FAIRE
   x ← x − 1
FIN TANT QUE

5.5 – Fonctions

FONCTION carre(x)
   RETOURNER x × x
FIN FONCTION

Exercices corrigés

Exercice 1 : Somme 1 à 100
somme ← 0
POUR i DE 1 À 100 FAIRE
   somme ← somme + i
FIN POUR
→ Résultat : 5050

Exercice 2 : Factorielle de 5
fact ← 1
POUR i DE 1 À 5 FAIRE
   fact ← fact × i
FIN POUR
→ Résultat : 120

Exercice 3 : Max de 3 nombres
LIRE a, b, c
SI a ≥ b ET a ≥ c → max ← a
SINON SI b ≥ c → max ← b
SINON → max ← c
AFFICHER max
